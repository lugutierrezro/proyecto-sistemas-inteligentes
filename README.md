# Sistemas Inteligentes - GUTIERREZ ROBLES LUIS GUSTAVO

Repositorio oficial del curso **Sistemas Inteligentes** de la Universidad César Vallejo.

- **Algoritmos**: Búsqueda, lógica, planificación

- **Proyectos**: Implementaciones prácticas del curso

## Tecnologías
- Python 3.10+
- Jupyter Notebook
- NumPy, Pandas, Scikit-learn
- TensorFlow/PyTorch (según clases)
- Matplotlib/Seaborn
